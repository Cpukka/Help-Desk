# Enterprise Help Desk & IT Service Management System

## Overview
A comprehensive, enterprise-grade Help Desk and IT Service Management System built with ASP.NET Core 9 and Next.js 15.

## Features
- 🔐 JWT Authentication & Authorization
- 🎫 Complete Ticket Management
- 📊 Real-time Dashboard with Charts
- 📚 Knowledge Base System
- 👥 User & Role Management
- 📝 Audit Logging
- 📧 Email Notifications
- 🔔 Real-time Notifications with SignalR
- 📈 Reports & Analytics
- 🌙 Dark Mode Support
- 📱 Responsive Design

## Tech Stack

### Backend
- ASP.NET Core 9
- Entity Framework Core
- SQL Server
- JWT Authentication
- SignalR
- AutoMapper
- FluentValidation
- Serilog

### Frontend
- Next.js 15
- TypeScript
- Tailwind CSS
- React Query
- React Hook Form
- Chart.js

## Prerequisites
- .NET 9 SDK
- Node.js 20+
- Docker (optional)
- SQL Server 2022+
